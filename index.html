Are all the elements in a single row? (just in a row, nothing else)
There's a few ways to achieve this
using normal flow, make all the element display: inline-block (or inline-flex, inline-grid, etc) - Pros: white space is maintained between elements - Cons: layout is controlled by each individual element, so it's all over the place Probably best in situations that are very text like (eg label + input maybe?) I personally never use this option
using grid works well when you know exactly how many things there are _can_ work when you don't know how many things there are (grid-auto-flow: column) but this usage is unusual
using flex - the simplest way to achieve this - probably my go-to
exception: horizontal scrolling. can this work with flex? or only grid? or only normal flow?
